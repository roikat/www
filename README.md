# WWW

My personal portfolio and digital garden built with [Astro](https://astro.build/). Styled with [Tailwind CSS](https://tailwindcss.com/) utility classes, and enhanced with [Preact](https://preactjs.com/) components for interactive functionality.

### Run the project

1. Download the dependencies

```bash
pnpm i
```

2. Launch the dev server

```bash
pnpm dev
```
